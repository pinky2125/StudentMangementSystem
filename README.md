# 🎓 Student Management System (Java)

## 📌 Description
This is a console-based Java application to manage student records.

## 🚀 Features
- Add Student
- View Students
- Update Student
- Delete Student
- Duplicate ID Check
- Input Validation
- Exception Handling

## 🛠️ Technologies Used
- Java
- OOP Concepts
- ArrayList
- Git & GitHub

## ▶️ How to Run
1. Open project in Eclipse
2. Run Main.java
3. Follow menu options
